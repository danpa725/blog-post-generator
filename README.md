## Run apps on the terminal without any configuration

-   Run once

    ```sh
    npx -y @danpacho/post-generator
    ```

-   Run with scripts

    add scripts to the `package.json`

    ```json
    {
        "scripts": {
            "post": "npx -y @danpacho/post-generator"
        }
    }
    ```

    and then

    ```sh
    yarn post
    ```
